hal
===

hal is a CNF-SAT solver based on the DPLL algorithm.  It takes (CNF)
DIMACS files as input.

Roadmap
-------

- [x] SAT-solver
- [ ] DIMACS parser
- [ ] Optimisations

Building
--------

Using GNU make:
```bash
$ make all
```
